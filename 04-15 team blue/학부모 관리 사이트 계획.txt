학부모 관리 시스템

필요한 기능
1. 회원가입 - 관리자, 교사, 학부모
2. 공지사항
3. 앨범
4. 알림장
5. 일정표

세부사항
관리자는 하나의 단체입니다. -> 관리자(학교에모든 권한 가짐, 관리자 생성 -> 테이블생성)
학교 관리자 = 단체 장 (학교 -> 교장, 유치원 어린이집 -> 원장)
                        ↓
교사 = 단체에 소속되어있는 교사 (관리자에게 귀속되어있는 직원)
                        ↓
학부모 = 교사에게 소속되어있는 학무모 (교사에게 귀속되어 있는 반 아동의 학부모)

ex)
1. 한국소프트웨어인재개발원 관리자 아이디 회원가입
강지웅 회원가입 한국소프트웨어인재개발원 관리자가 부여한 코드 입력시 교사로 회원가입 가능
공현 회원가입 어디로할지 검색 -> 한국소프트웨어 인재개발원 검색 -> 강사 강지웅 선택 -> 강지웅 소속 학생으로 가입 완료(원래는 학부모가 가입해야함)

2. 한국소프트웨어인재개발원 관리자 공지사항 작성 -> 강지웅, 공현 확인가능
강지웅 공지사항 작성 -> 공현 확인가능
공현 공지사항 작성 불가

3. 관리자, 강지웅 앨범 작성 가능(관리자는 작성할 일이 없긴함 작성이 가능하긴 함)
공현 앨범 작성 불가

4. 관리자, 강지웅, 공현 다 알림장 작성가능(역시나 관리자는 작성할일 없음 작성은 가능 함)
교사(1) 학부모(다) 형식으로 작성
강지웅이 작성한 알림장은 공현과 강지웅 확인가능

공현이 작성한 알림장 공현과 강지웅 확인가능하지만 강지웅이 가르치는 다른 학생의 학부모는 확인 불가능

5. 일정표는 관리자, 강지웅 작성 가능
모두가 확인 가능.

6. 글 댓글 권한은 모두에게 있음.

7. 글 삭제 권한은 본인이 작성한 글, 하위 그레이드에서 작성한 글 삭제 가능 ex) 강지웅 글 관리자가 삭제 가능, 공현 글 강지웅이 삭제 가능, 관리자 글 공현이 삭제 불가능

구현을 위해 필요한 것.

1. 회원가입
처음에 관리자, 교사, 학부모 가입중 택1
    관리자 가입시 기관 고유번호 입력
    교사 가입시 기관에서 부여한 기관 교사가입가능 고유번호 입력
    학부모는 그냥 검색해서 가입 가능
개인정보(이름, 아이디, 비밀번호, 주민번호)
네이버, 카카오 등 간편로그인 구현 가능한지 확인해서 구현할것.

2번 이후 권한은 위에 설명 함.




관리자, 교사 사이트와 학부모 사이트 분리
처음 로그인시 권한을 체크하여 관리자교사 사이트, 학부모 사이트로 이동

관리자, 교사 사이트는 우리가 공부한대로 탭으로 들어가서 글 작성하는식으로 생성

학부모사이트는 본인이 볼 수 있는 글만 피드로 받아서 화면에 띄움
